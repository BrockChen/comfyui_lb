# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.0.1] - 2025-12-21

### Added
- Initial release of ComfyUI plugin for Dify
- Support for ComfyUI workflow API integration
- Automatic image generation and download
- Support for ComfyUI load balancer
- Async processing with timeout handling
- Credential validation for ComfyUI connection
- Comprehensive error handling and reporting

### Features
- Generate images using ComfyUI workflow API JSON
- Automatic polling for task completion
- Image blob storage in Dify
- Support for both standalone ComfyUI and load balancer setups

### Documentation
- README with installation and usage instructions
- Example workflow JSON
- Publishing guide for GitHub and Dify Marketplace
